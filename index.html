About

The person behind
the code

I'm Mohammad Yasir Khan, a MERN stack developer from Lucknow, India. I got into web development because I love the intersection of logic and craft — there's something deeply satisfying about a well-structured API and a smooth UI working together seamlessly.

On the backend, I'm comfortable designing Express.js REST APIs with proper auth (JWT access/refresh token rotation, httpOnly cookies), handling file uploads with Multer, and integrating image CDNs like Cloudinary and ImageKit. I think seriously about system design — rate limiting, token security, separation of concerns.

On the frontend, I reach for React with Zustand for state, React Router v6 for navigation, and polish UIs with Framer Motion and DaisyUI. Lately I've been excited about integrating Gemini AI with structured outputs — using Zod to make AI responses actually reliable in production.

I also script in Python — using uv for fast project setup and requests for HTTP automation and data pipelines.
